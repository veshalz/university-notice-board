# University Notice Board

A full-stack web application designed to digitize campus communications, manage student assignments, and track academic events.

## 🚀 Key Features
* **Dynamic Dashboard:** Real-time feed for urgent notices, academic events, and department announcements.
* **Campus Voice:** Interactive polling system to gather student feedback on campus initiatives.
* **Smart Bookmarks:** Users can pin important academic resources for quick access.
* **Responsive Design:** Optimized for mobile and desktop, ensuring accessibility for all students.
